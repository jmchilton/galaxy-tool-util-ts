    Class ToolCache

    Two-layer cache (memory + storage) for parsed Galaxy tool metadata. Resolves tool IDs to cache keys, loads/saves tool JSON, and manages the cache index.

    storage is required — pass a FilesystemCacheStorage (Node.js) or IndexedDBCacheStorage (browser). Node callers can use makeNodeToolCache from @galaxy-tool-util/core/node to get the default filesystem setup.
