    refetch

    • refetch(
          toolId: string,
          toolVersion?: string | null,
          opts?: { force?: boolean },
      ): Promise<{ alreadyCached: boolean; cacheKey: string; fetched: boolean }>

      Populate the cache for a known tool id. Idempotent by default — when the entry already exists, returns {alreadyCached: true, fetched: false} and skips the remote fetch. Pass {force: true} to evict the existing entry first and re-fetch unconditionally.

      Used by inspector / debug surfaces (gxwf-web /api/tool-cache/{add,refetch}).

      Parameters

      • toolId: string
      • OptionaltoolVersion: string | null
      • Optionalopts: { force?: boolean }

      Returns Promise<{ alreadyCached: boolean; cacheKey: string; fetched: boolean }>
